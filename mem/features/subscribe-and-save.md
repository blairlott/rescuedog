---
name: Subscribe & Save terminology
description: S&S is recurring auto-ship, NEVER called a "club". Wine Club / The Pack is a separate membership tier.
type: constraint
---
**Subscribe & Save (S&S / auto-ship)** is a recurring fulfillment convenience for a specific SKU/bundle. It is NOT a club.

**Wine Club / The Pack** is the membership tier (curated allocations, member pricing, perks).

They can coexist (a Pack member can also S&S a SKU) but the words stay distinct in all UI copy, emails, CRM labels, and code identifiers.

**Why:** Calling S&S a "club" confuses it with the actual Wine Club and dilutes both products.

**How to apply:** Use "Subscribe & Save", "subscription", "auto-ship", or "recurring order". Never "S&S club", "subscription club", or similar.
